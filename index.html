구문해석 기초 (문장의 형식)

문장의 기본 뼈대인 1형식부터 5형식까지의 구조와 품사의 역할을 이해합니다.

  • 1형식: S + V (주어 + 동사) + M(수식어)
  • 2형식: S + V + SC (주어 + 동사 + 주격보어)
  • 3형식: S + V + O (주어 + 동사 + 목적어)
  • 4형식: S + V + IO + DO (주어 + 동사 + 간접목적어 + 직접목적어)
  • 5형식: S + V + O + OC (주어 + 동사 + 목적어 + 목적격보어)

✂️ 끊어읽기 연습

1. The doctor / wrote / the patient / a prescription / for his disease.
(의사는 / 써주었다 / 환자에게 / 처방전을 / 그의 질병을 위한)

2. Abilities like empathy and communication / make / us / human.
(공감과 소통 같은 능력들은 / 만든다 / 우리를 / 인간답게)

3. Surfing the internet / gives / me / a lot of information.
(인터넷을 검색하는 것은 / 준다 / 나에게 / 많은 정보를)

4. His grandfather / bought / him / a new bicycle.
(그의 할아버지는 / 사주셨다 / 그에게 / 새 자전거를)

5. The teacher / considered / the boy / a genius.
(선생님은 / 여겼다 / 그 소년을 / 천재로)

✅ 확인 문제

Q. 다음 문장의 형식을 고르시오.
"Governments have a responsibility to protect culture."

정답: 3형식 (S + V + O)
해설: 주어(Governments) + 동사(have) + 목적어(a responsibility) 구조이며 뒷부분은 수식어구입니다.

to부정사와 동명사

동사가 명사, 형용사, 부사 등으로 변신하여 문장에서 다양한 역할을 합니다.

  • 명사적 용법: 주어, 목적어, 보어 역할 (~하는 것)
  • 형용사적 용법: 앞의 명사를 수식 (~할, ~하는)
  • 부사적 용법: 목적, 결과, 원인 등을 나타냄 (~하기 위해, ~해서)
  • 의문사 + to부정사: 명사구 역할, 의문사 + 주어 + should + 동사원형으로 변환 가능

✂️ 끊어읽기 연습

1. I don't know / how to allocate the budget / fairly.
(나는 모른다 / 예산을 어떻게 할당할지 / 공정하게)

2. She couldn’t decide / what to say / at the meeting.
(그녀는 결정할 수 없었다 / 무엇을 말할지 / 회의에서)

3. I have / some homework to finish / by tomorrow.
(나는 가지고 있다 / 끝내야 할 약간의 숙제를 / 내일까지)

4. To master a language / requires / constant practice.
(언어를 마스터하는 것은 / 요구한다 / 끊임없는 연습을)

5. He went to the store / to buy some milk.
(그는 가게에 갔다 / 우유를 약간 사기 위해)

✅ 확인 문제

Q. 다음 중 형용사적 용법으로 쓰인 to부정사는 무엇일까요?

1. To read books is fun.
2. I need something to drink.
3. I went there to meet him.

정답: 2번
해설: something이라는 대명사를 뒤에서 '마실'이라는 형용사처럼 수식하고 있습니다. 1번은 명사적, 3번은 부사적(목적) 용법입니다.

동명사와 분사구문 완전 정복

동명사와 to부정사의 본질적 의미 차이와 분사구문의 변환 원리를 다룹니다.

  • 동명사(-ing): 과거 지향 (이미 일어난 일, 경험, 습관)
  • to부정사(to V): 미래 지향 (아직 안 일어난 일, 소망, 의지)
  • 분사구문 변환: 접속사 생략 → 주어 일치 시 생략 → 동사를 V-ing로 변환
  • 분사구문의 부정: Not 또는 Never를 분사구문 맨 앞에 위치시킴 (예: Not knowing)

✂️ 끊어읽기 연습

1. Not knowing the answer, / he kept silent.
(정답을 알지 못해서, / 그는 침묵을 유지했다)

2. I remember / locking the door / before leaving.
(나는 기억한다 / 문을 잠갔던 것을 / 떠나기 전에)

3. Seen from a distance, / the rock / looked like / a human face.
(멀리서 보였을 때, / 그 바위는 / ~처럼 보였다 / 사람의 얼굴)

4. Opening the envelope, / I found / a letter inside.
(봉투를 열었을 때, / 나는 발견했다 / 안에 있는 편지를)

5. Feeling tired, / she went to bed / early.
(피곤함을 느껴서, / 그녀는 잠자리에 들었다 / 일찍)

✅ 확인 문제

Q. 괄호 안의 올바른 형태를 고르시오.
"I stopped ( smoking / to smoke ). (해석: 나는 담배를 끊었다)"

정답: smoking
해설: 'stop + -ing'는 하던 행위를 그만두다(과거 지향)라는 뜻입니다. 'stop + to V'는 ~하기 위해 멈추다라는 의미가 됩니다.

주어 해석하기

다양한 형태의 명사류가 주어로 올 때 단수 취급을 하며 덩어리로 묶어 해석하는 것이 중요합니다.

  • 명사구 주어: to부정사, 동명사 (~하는 것은)
  • 명사절 주어: that, whether, 의문사가 이끄는 절 (S가 V하다는 것은 / S가 V하는지는)
  • 가주어 it: 주어가 너무 길 때 주어 자리에 it을 쓰고 진주어는 뒤로 보냅니다.
  • 단수 취급: 구나 절이 주어로 쓰일 때는 항상 3인칭 단수 취급합니다.

✂️ 끊어읽기 연습

1. Whether this children's song will be a big hit / is / doubtful.
(이 동요가 크게 유행할 것인지 아닌지는 / ~이다 / 의심스러운)

2. Where we spend our vacation / determines / the total expenditure.
(우리가 어디서 휴가를 보낼지가 / 결정한다 / 총 지출을)

3. To make new friends / is / not always easy.
(새로운 친구를 사귀는 것은 / ~이다 / 항상 쉽지만은 않은)

4. That he loves her / is / true.
(그가 그녀를 사랑한다는 것은 / ~이다 / 사실)

5. It is important / for us / to protect the environment.
(중요하다 / 우리가(의미상 주어) / 환경을 보호하는 것은)

✅ 확인 문제

Q. 다음 문장의 빈칸에 알맞은 단어를 고르시오.
"Learning some phrases in Spanish ( is / are ) helpful."

정답: is
해설: 동명사구(Learning ~ Spanish)가 주어이므로 3인칭 단수 취급하여 is를 씁니다. phrases에 속아 복수로 쓰면 안 됩니다!

수능 영어 영역별 스킬 & 공부방법

수능 영어의 대표적인 고난도 유형들을 정복하기 위한 전략과 올바른 접근법을 소개합니다.

1. 대의 파악 (주제, 제목, 요지)

  • 접근 스킬: 첫 문장과 마지막 문장을 꼼꼼히 읽어 글의 방향성을 잡습니다. 'However', 'But', 'Therefore'와 같은 연결사 뒤의 문장에 글쓴이의 주장이 담겨있는 경우가 많습니다.
  • 공부 방법: 지문을 다 읽은 후, 스스로 한 문장으로 글의 내용을 요약해보는 연습을 꾸준히 하세요. 선택지를 읽기 전에 먼저 우리말로 정답을 예측해 보는 것이 좋습니다.

2. 빈칸 추론

  • 접근 스킬: 빈칸은 절대 지엽적인 내용에 뚫리지 않습니다. 항상 글의 핵심 주제(Main Idea)와 직결되거나, 주제를 뒷받침하는 핵심 근거에 빈칸이 존재합니다. 빈칸 문장을 먼저 읽고 '무엇을 찾아야 하는지' 타겟을 설정한 후 지문을 읽으세요.
  • 공부 방법: 지문 속에 반드시 빈칸에 들어갈 내용과 똑같은 의미를 지닌 '재진술(Paraphrasing)' 문장이 숨어 있습니다. 같은 의미를 다르게 표현한 단어와 구를 연결하는 훈련을 하세요.

3. 순서 배열 & 문장 삽입

  • 접근 스킬: '지시어(This, That, Such)', '대명사(He, They, It)', '연결사(For example, On the other hand)'라는 3대 힌트를 적극적으로 활용해야 합니다. 내용적 흐름뿐만 아니라 논리적 단서를 찾아 꼬리물기를 하듯 연결합니다.
  • 공부 방법: 정답을 맞힌 후에도 왜 A 다음에 B가 와야 하는지 근거에 밑줄을 긋는 연습을 하세요. 근거 없이 감으로 푸는 습관을 고치는 것이 가장 중요합니다.

4. 어법 & 어휘

  • 접근 스킬: 수능 어법은 나오는 패턴이 정해져 있습니다. (수일치, 동사/준동사 구별, 관계대명사/접속사 구별 등) 어휘 문제는 반의어가 정답인 경우가 90% 이상이므로 긍정(+)/부정(-) 뉘앙스를 파악하세요.
  • 공부 방법: 기출문제에 나온 어법 포인트들을 노트에 정리하고, 각 탭 1~4에서 배운 '구문 분석(주어, 동사 찾기)'을 철저히 연습하여 문장 구조를 한눈에 파악하는 능력을 기르세요.
